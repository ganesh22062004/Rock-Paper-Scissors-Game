# ✊✋✌️ Rock Paper Scissors Game

A simple and interactive **Rock Paper Scissors Game** built using **Python**. This project allows users to play the classic Rock-Paper-Scissors game against the computer through a command-line interface. The computer makes random choices, and the game determines the winner based on the standard rules.

---

## 📌 Features

- 🎮 Play against the computer
- 🎲 Computer selects a random choice
- 🏆 Displays Win, Lose, or Tie result
- 🔄 Play multiple rounds
- ❌ Input validation for invalid choices

---

## 🛠️ Technologies Used

- **Python 3**
- Random Module
- Conditional Statements
- Loops
- Command-Line Interface (CLI)

## 🚀 Getting Started

### Prerequisites

- Python 3.8 or later

### Installation

1. Clone the repository:

```bash
git clone https://github.com/ganesh22062004/Rock-Paper-Scissors-Game.git
```

2. Navigate to the project directory:

```bash
cd Rock-Paper-Scissors-Game
```

3. Run the application:

```bash
python rock_paper_scissors.py
```

---

## 📖 How It Works

After running the program, the player is prompted to choose one of the following:

```text
Rock
Paper
Scissors
```

The computer randomly selects its choice, and the winner is determined using the standard Rock-Paper-Scissors rules.
